Auto-pauses all proxy traffic for the rest of the month once this much data is used, a backstop that protects your Cloudflare account from abuse-detection suspensions. 0 = no cap.

Speed limit

Caps the total download speed of each user across all their connections, a fairness backstop so no single user can saturate the worker. 0 = unlimited. A per-user limit (on the Users page) overrides this.

📘 How it works

Control all your Nova panels from this one. Add your other panels below, and this panel copies its settings, routing and clean IPs to them every time you save.

1
Add each panel
Press “+ Add a panel” and enter its name, its web address, and its admin password (the same one you log in with on that panel).
2
Save and sync
Press “Save & sync all”. Your settings are copied to every panel. It also syncs on its own each time you save here.
💡 Good to know
  • ⚠️ Syncing replaces the other panel’s settings with this one’s, including its ID, hosts and users. Only add panels you want to become copies of this hub.
  • Nothing to set up on the other panels. Just add them here with their password.
  • Each panel must be on its own Cloudflare account. A panel cannot control another one on the same account.
  • Keep every panel on the same Nova version (open /install/status on each to check).
  • If a panel’s admin password changes, update it here too.

🛡️ Cloudflare CDN access

          

            
How the Worker reaches Cloudflare's network. Choose a mode, enter the proxy, hit Verify to probe it, then Save. This is the single source of truth, configs pull from here.

            

              

            

            

            

              
One proxy per line. One = fixed/static exit IP (good for Instagram); two-three = the app rotates and fails over between them. · یک پروکسی در هر خط؛ یکی = آی‌پی ثابت، دو تا سه = چرخش و جایگزینی خودکار.

              

                

                

              

              
With several proxies above, Auto-rotate serves a different subset each day/week, the exit set changes automatically and clients pick it up on their next subscription refresh. · با چند پروکسی، هر روز/هفته زیرمجموعه‌ای متفاوت ارائه می‌شود.
